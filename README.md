# Voyage — Tourism Web Application

A full-featured tourism web application built with Django.

## Features
- Browse and search travel tours with filters
- View detailed tour descriptions
- Add tours to favorites or cart
- Simulated ticket purchase flow (cart → buy → confirmation → "My Tours")
- User registration and login
- Personal profile with avatar and bio
- Admin panel for managing tours and users

## Tech Stack
- Python, Django
- SQLite
- HTML, CSS

## How to Run
```bash
git clone https://github.com/Lemoonchoy/diploma
cd diploma
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
